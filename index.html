📘 Révisions — Les nombres relatifs

Comparer des nombres relatifs

Plus un nombre est éloigné de 0 du côté négatif, plus il est petit.
Exemple : -12 < -3 < 0 < 5.

Valeur absolue et opposé

Valeur absolue : distance à 0 (toujours positive).
Opposé : même distance à 0 mais signe contraire.

Addition de relatifs

➕ Même signe → on additionne les distances et on garde le signe.
➕ Signes différents → on soustrait les distances et on garde le signe du plus grand.

Soustraction de relatifs

Soustraire un nombre relatif revient à ajouter son opposé.

Multiplication de relatifs

✖ Deux nombres de même signe → positif.
✖ Deux nombres de signes contraires → négatif.
✖ Plusieurs facteurs :
- Pair de négatifs → produit positif.
- Impair de négatifs → produit négatif.

Division de relatifs

➗ Même règle de signe que la multiplication :
- Signe identique → quotient positif.
- Signe contraire → quotient négatif.
